MCGI Attendance Home Screen Icons

Files:
- mcgi-icon-apple-180.png: iPhone/iPad Add to Home Screen icon
- mcgi-icon-192.png: standard Android/web icon
- mcgi-icon-512.png: high-resolution Android/web icon
- mcgi-icon-maskable-512.png: Android maskable icon
- mcgi-icon-master-1024.png: master archive copy
- manifest.webmanifest: Android web app manifest template

Before uploading:
1. Replace REPLACE_WITH_YOUR_APPS_SCRIPT_EXEC_URL in manifest.webmanifest.
2. Upload all files to the same GitHub Pages folder.
3. Update Index.html references.
4. Redeploy Apps Script.
5. Delete the old Home Screen shortcut and add it again.
